• Tạo một tài khoản với github: https://github.com/
  • Tạo repo: - trở vào New bên trái màn hình, đặt tên và tạo:
  • sao khi tạo thành công, sẽ có 1 đường link HTTPs: Ex: https://github.com/leykhoa/bai_13.git
  • Mở vscode chưa thư mục code
  • Terminal (Ctrl + `)
  • sử dụng câu lệnh: git init (dùng để khởi tạo git)
  • Thêm các file sửa đổi vào commit: git add .
  • Commit git: git commit -m "name commit"
  • Thêm đường dẫn của repo: git remote add origin https://github.com/leykhoa/bai_13.git
  • Push nội dung lên online: git push -u origin master
